# MD-LP-noise-robustness-evaluation
This code is the experimental code used in the paper *On Evaluating Noise Robustness of Deep Networks via Minkowski-Difference Linear Probe*. The dataset used in this code is CIFAR-10 and UTKFace. The environment used for this code is TensorFlow 2.7 and Python 3.7.0.

The code is divided into two folders. The code in the "noise" folder is used to evaluate the noise robustness of the network that was trained without noise-imposed samples. The "noise-2" folder is used for evaluating the noise robustness of the network that was trained with samples with imposed noise.
